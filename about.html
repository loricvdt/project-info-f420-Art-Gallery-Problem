The Art Gallery Problem

INFO-F420: Computational Geometry project by Loric Vandentempel

< Back to project

This web-based project is programmed in JavaScript using the p5 library. All the illustrations are therefore rendered using a custom-made engine for p5 with features that can be enabled or disabled.

The main features of the engine are:

  • Adding points successively by clicking
  • Turning that point list into a polygon
  • Importing a pre-set polygon
  • Adding points representing the guards position
  • Displaying the field of view of every placed guard
  • Resetting all or parts of the canvas

All these features implement several checks to ensure proper use of the interactive features. First it prevents the addition of an edge to the polygon if it intersects another existing one. Once the polygon is fixed (i.e. when all the points are connected), the guard points can be added but only on the inside of the polygon.

The field of view of each guard is calculated by checking which vertex is directly in sight of the guard by checking for intersection of all the other polygon edges. Then, with this list of vertices in sight it is easy to check which pair of consecutive points belong to an edge of the polygon. The line of sight of the guard goes between the pairs that do not form an edge, the closest point of intersection from the guard to the edge can be calculated, giving the limit of their vision.

The entirety of the project makes use of tools learned during the Computational Geometry course and its homeworks. These concepts include: the orientation determinant, polygons, point location, intersections, and ray shooting.
